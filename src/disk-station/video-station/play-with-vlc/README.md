# play-with-vlc

DiskStationのVideo Stationで動画をVLCで再生します。

## 要件

- `diskstation.local` がDiskStation Managerに割り当てられている
- VLCのためのURLスキーム (`vlc://<file-path>`) が使用可能
- DiskStationのビデオディレクトリがローカルディスクにマウントされている

## 使い方

1. [play-with-vlc.user.js](https://github.com/munierujp/userscripts/raw/master/dist/disk-station/video-station/play-with-vlc.user.js)をインストール
